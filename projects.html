Predicting SLE-Associated Epitopes with Deep Learning

Python PyTorch Bioinformatics
View on GitHub

Built a deep learning model to predict the immunogenic potential of epitopes linked to systemic lupus erythematosus (SLE), using 6,553 peptides from the IEDB epitope database.

  • Reimplemented MITNet, a state-of-the-art epitope prediction architecture, then extended it with biologically informed features beyond sequence data (e.g., structural and chemical attributes).
  • Designed a new transformer-based architecture to incorporate multimodal peptide representations, boosting F1-score to 0.94 — a 32% improvement over the MITNet.
  • Conducted comparative evaluation with ProtBERT, ReLSO, and BEETLE; validated results using stratified cross-validation and ROC analysis.
