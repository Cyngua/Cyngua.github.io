LLM-Powered Metadata Extraction for Scientific Code Repositories

Python LLM OpenAI API spaCy Pandas Prompt Engineering API Integration
View on GitHub

Built an automated metadata extraction pipeline to tag and classify 1,800+ computational neuroscience models from ModelDB, working on code-heavy, non-natural language inputs.

  • Engineered a baseline rule-based system using 5-gram keyword search and regex logic to extract 7 metadata types (e.g., neuron cell types, ion channels, simulation environments).
  • Enhanced extraction accuracy and consistency using GPT-4 with chain-of-thought prompting; achieved significant improvements in precision and F1 scores across metadata classes, with over 0.5 Jaccard consistency.
  • Integrated spaCy and Pandas in a preprocessing stage to isolate relevant tokens before model inference.
  • Automated tagging for new submissions and streamlined model discoverability for researchers through integration with ModelDB's metadata infrastructure.

Clinical Entity Recognition & Ontology-Aware NLP Pipeline

Python HuggingFace Transformers UMLS API MIMIC-IV HPC (GPU)
View on GitHub

Developed a clinical NLP pipeline to extract and classify semantic groups from deidentified MIMIC clinical notes using SNOMED-CT annotations and UMLS semantic tags.

  • Fine-tuned a ClinicalBERT model with a CRF layer on GPU-based high-performance computing resources to improve span-level precision; achieved F1 score of 0.75 on test set.
  • Integrated UMLS API to normalize entity labels into ontology-aligned categories (e.g., condition, procedure, drug), enabling downstream compatibility and interpretability.
  • Delivered as a modular pipeline and command-line application for internal clinical teams, designed for future integration into EHR decision support systems.
